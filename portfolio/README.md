# kileymeehan.com

Personal site. Static HTML/CSS/JS — no build step required.

## Dev

Open `index.html` in a browser, or run a local server:

```bash
npx serve .
```

## Deploy

Drop the folder contents onto your host (Netlify, Vercel, GitHub Pages, etc.).
No build step needed.

## Key files

| File | Purpose |
|------|---------|
| `index.html` | Main page |
| `css/style.css` | All styles |
| `js/main.js` | Minimal JS |
| `assets/` | Images — add `hero-photo.jpg` here |
| `CLAUDE.md` | Design brief for Claude Code |
